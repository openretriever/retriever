# Retriever Hub

Retriever has three public surfaces:

- **Core runtime**: Flow, Clock, Sync Policy, Pipeline, stepping, replay, IR, and backend execution.
- **Retriever Hub**: the reusable module protocol and index for Flows, type packs, transforms, and composed pipelines.
- **GoldenRetriever**: the first applied robotics Hub module and companion example catalog.
